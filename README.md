# 🎮 Questify

Questify é um sistema simples em Java que transforma tarefas do dia a dia em missões gamificadas.

## 🚀 Funcionalidades atuais

* Criação de usuário
* Criação de missões
* Sistema de dificuldade:

    * Fácil
    * Média
    * Difícil
    * Épica
* Sistema de recompensas:

    * XP
    * Moedas
* Conclusão ou cancelamento de missões
* Sistema de nível do usuário

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Encapsulamento
* Construtores
* Enum
* Estruturas condicionais (switch)
* Entrada de dados com Scanner
